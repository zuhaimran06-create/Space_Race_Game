# Space Race Game

A two-player racing game built using Python and Pygame. Players race to the finish line while avoiding moving obstacles.

## Features

- Two-player gameplay
- Collision detection
- Score tracking
- Start screen and play-again screen
- Animated winner sequence
- No external image or music files required

## Controls

### Player 1
- W: Move up
- S: Move down

### Player 2
- Up Arrow: Move up
- Down Arrow: Move down
